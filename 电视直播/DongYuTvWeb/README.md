## 📢 下载与安装须知（设备版本选择指南）

安装包区分**32位和64位**，请下载对应的CPU架构进行下载，**老旧设备请使用X5内核版本**。如果不能播放可都下载试试。

| 版本类型      | 适用设备及说明                                     | 支持架构 |
| :------------ | :------------------------------------------------- | :------: |
| **WebView版** | 适合 Android 版本较高的手机或设备。体积较小。      |   64位   |
| **X5内核版**  | 专为电视或者部分老旧安卓设备优化，内置腾讯X5内核。 |   64位   |
| **X5内核版**  | 专为电视或者部分老旧安卓设备优化，内置腾讯X5内核。 |   32位   |
